SDG 1: No Poverty Application
===============================

A full-stack application to address SDG 1: No Poverty through financial empowerment, 
crowdfunding, microloans, and AI-powered financial guidance.

Features:
- User authentication and profile management
- AI-powered budgeting recommendations
- Transaction tracking and analysis
- Crowdfunding campaigns
- Microloan applications and tracking
- Interactive poverty map
- Modern responsive UI

Tech Stack:
- Backend: FastAPI (Python)
- Frontend: HTML/CSS/JS with Bootstrap and Chart.js
- Database: PostgreSQL
- AI: scikit-learn for financial analysis
- Deployment: Heroku
